August

In August, I began Communication Design. We didn't really do much except learn the basics of a good website. We compared what made a website good and what made a website bad. We also briefly went over how to create a rubric for a good website.

September

In September, we continued to learn about the basics of what it takes to make a website good. We also teamed up in groups to, one, create a rubric for what makes a website good and, two, to learn how to organize a website. In addition, we learned how to create the "foundation" of a website by coding in HTML. We learned some of the coding in class and some of the coding from the website Codecademy. We learned what semantic tags were, how to tag, how to start coding a website, and how to make our website attractive. The most important thing we learned was how to make a webiste accessible to everyone.
