Semantic Tags

  • <nav>: Used for the website navigation
  • <main>: Used for the main content of a page
  • <section>:Defines a section in a docoument. A section is a thematic grouping of content. Example:
    • Introdution
    • Content
    • Contact Informaiton
  • <article>: Specifies indepent and sef-contained content
  • < footer>: Specifies a footer for a document or section
  • < aside>: Defines content aside from the content it is placed in (e.g.sidebar)

Formatting tags

There are several content tags that are used for formatting text. There tags are not block tags. This means that they are used inline to format text inside a block.
We can make text bold, strong, italic , or emphasized. Other ways of emphasizing text include highlighting it or underlining it. We can also change text to a smaller size, write it as a subscript , a superscipt or crossed out .
